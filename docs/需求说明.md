其他说明：
1、关于cvd曲线分析部分，标准化为Z-Score（最推荐）
这是最常用和最科学的方法，它能最好地反映“相对动能”。
操作步骤：
对每个标的，取其CVD时间序列数据。
计算该CVD序列的均值 和标准差。
对序列中的每一个数据点进行Z-Score标准化： (当前CVD值 - 均值) / 标准差
将标准化后的数据绘制在同一张图中。
优点：
无单位，可比较：所有数据都变成了“偏离均值多少个标准差”的单位。你可以直接比较一个股票偏离其自身正常水平的程度。Z-Score为2的股票，其资金推动强度远大于Z-Score为0.5的股票，无论它们本身的市值大小。
清晰显示动能：曲线在零轴上方表示资金净流入强于平均水平，下方则表示弱于平均水平。曲线的斜率就代表了动能的加速或减速。
可视化示例：
Y轴：CVD (Z-Score)
X轴：时间
这样我在图中寻找的是：
持续位于高位（如Z-Score > 1）且向上倾斜的曲线 -> 强势买入动能
持续位于低位（如Z-Score < -1）且向下倾斜的曲线 -> 强势卖出动能
在零轴附近徘徊的曲线 -> 动能中性，多空平衡
