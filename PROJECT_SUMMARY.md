### 2. 核心算法

#### Z-Score标准化
```python
Z-Score = (CVD - 均值) / 标准差
```
- 消除量纲影响，实现跨标的比较
- 标准化到统一尺度（均值为0，标准差为1）
- 直观显示偏离程度

#### 背离检测算法
```python
背离条件：
- CVD极值 (|Z-Score| > 1.0)
- AND 价格变化趋势相反 (> 5%)
```

### 3. 性能优化
- **数据缓存**: `@st.cache_data` 缓存60秒
- **增量处理**: 只加载必要数据
- **响应式设计**: 适配不同屏幕
- **内存优化**: pandas向量化操作
